

# Springboot-MetaMall-Project

MetaMall은 사용자가 상품을 찾아보고, 구매하고, 관리할 수 있는 전자상거래 토이 프로젝트 입니다.

## 기능

**사용자 인증**

* 사용자 등록 및 로그인 기능

* 고객과 판매자를 위한 역할 기반 접근 제어

**상품 관리**

* 상품 등록, 수정, 삭제, 목록 보기

**주문 관리**

* 주문 등록, 취소, 목록 보기

**에러 처리**

* 모든 에러 로그기록
